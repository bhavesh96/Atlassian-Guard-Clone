## Atlassian Guard Clone

A React + Tailwind CSS front-end project that replicates the Atlassian Guard landing page.
This project showcases modular React components, reusable UI buttons, and responsive design built with Tailwind CSS.

## Components Overview

| Component | Description |
|------------|-------------|
| **Navbar** | Displays the navigation bar and site logo |
| **HeroSection** | Main banner with heading, tagline, and primary CTA button |
| **MeetGuardSection** | Section introducing *Guard Premium* with illustrations |
| **PdrSection** | Interactive section with *Protect, Detect, Respond* buttons and video previews |
| **Footer** | Responsive footer with product links, company info, and legal links |


## Technologies Used

- **React.js** — Component-based UI library  
- **Tailwind CSS** — Utility-first CSS framework  
- **JavaScript (ES6+)**  
- **HTML5 Video & SVG assets**
- **Responsive Design**


## Features

-  Clean and modular React components  
-  Fully responsive design (mobile-first)  
-  Reusable button & section components  
-  Dynamic rendering using React `useState`  
-  Smooth hover and transition effects  
-  Fast build and hot reload via Vite
